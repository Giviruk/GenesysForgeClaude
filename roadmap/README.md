## Общая продуктовая логика

GenesysForge должен развиваться не как очередной онлайн-лист персонажа, а как инструмент подготовки и ведения кампаний Genesys / Realms of Terrinoth.

Главная цепочка мастера:

1. Создать кампанию.
2. Собрать Campaign Handbook.
3. Создать NPC в Adversary Library.
4. Собрать Encounter.
5. Отправить Encounter в Game Table.
6. Провести сцену через GM Cockpit.
7. Распечатать карточки при необходимости.

Главная цепочка игрока:

1. Создать персонажа.
2. Присоединиться к кампании.
3. Читать Handbook.
4. Использовать лист персонажа.
5. Собирать магические действия.
6. Смотреть Game Table во время сцены.
7. Печатать карточки своих талантов, предметов и заклинаний.
