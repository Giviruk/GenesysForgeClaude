# Справка GenesysForge

Этот раздел помогает пройти базовый сценарий без README: создать персонажа, вести развитие, собрать сцену для ведущего и понимать, какие данные показываются в разных режимах контента.

GenesysForge не заменяет книги правил. В приложении используются структурные данные, числовые параметры и краткие собственные описания, без оригинальных текстов из официальных книг.

## Быстрый маршрут игрока

- Откройте «Персонажи» и нажмите «Новый персонаж».
- Выберите систему: Genesys Core или Realms of Terrinoth.
- Выберите архетип/расу и карьеру.
- Отметьте свободные карьерные навыки, если мастер разрешил стандартный стартовый набор.
- Проверьте доступный XP и завершите создание, когда стартовый лист готов.

Во время создания часть покупок можно отменять. После завершения создания возвраты стартовых покупок становятся недоступны: дальнейшее развитие идёт через выдачу XP и новые покупки.

## Характеристики, навыки и карьерные навыки

- Характеристики покупаются только в фазе создания.
- Навыки покупаются за XP. Карьерные навыки дешевле, не карьерные стоят дороже.
- Свободные карьерные ранги, полученные при создании, не считаются покупкой за XP и не возвращаются.
- Dice pool считается автоматически по характеристике и рангу навыка.

Если кнопка покупки недоступна, обычно причина в нехватке XP, лимите фазы создания или правиле максимального ранга.

## Таланты и пирамида

Таланты покупаются по tiers. Для более высоких tiers нужно поддерживать пирамиду: нижних талантов должно быть достаточно, чтобы верхний уровень оставался валидным.

Если талант ranked, повторная покупка считается следующим эффективным tier. Если талант не ranked, его можно купить только один раз.

## Инвентарь и производные характеристики

- Добавляйте предметы во вкладке «Инвентарь».
- Для брони и оружия важно состояние предмета: equipped, carried или backpack.
- Экипированная броня влияет на soak и defence по правилам приложения.
- Нагрузка и порог нагрузки пересчитываются автоматически.
- У оружия в инвентаре показывается dice pool для навыка этого оружия, если у персонажа есть соответствующий навык.

Если производная характеристика не изменилась, проверьте количество предмета и состояние «Экипировано».

## Магия

Страница «Магия» показывает действия, доступные для выбранной системы. В листе персонажа вкладка «Магия» использует навыки персонажа и показывает его dice pool.

Выберите базовое магическое действие, добавьте дополнительные эффекты и проверьте итоговую сложность. Карточку действия можно открыть в режиме печати.

## Кампании и присоединение игроков

Ведущий создаёт кампанию и передаёт игроку join code. Игрок присоединяет к кампании одного из своих персонажей. После этого ведущий может открыть лист игрока из кампании, а игрок видит доступные материалы кампании.

## NPC и бестиарий

В «Бестиарии» можно создавать NPC, дублировать существующие карточки и использовать их в энкаунтерах. Дублирование удобно для подготовки вариантов одного противника без изменения исходной карточки.

Для NPC доступны характеристики, навыки, атаки, способности, теги и видимость. Не копируйте в пользовательские карточки оригинальные тексты из книг; используйте собственные краткие описания.

## Энкаунтеры и Game Table

- В кампании откройте «Энкаунтеры».
- Создайте энкаунтер, добавьте NPC и персонажей игроков.
- Настройте видимость описания, цели сторон, угрозу и заметки ведущего.
- Нажмите «Отправить на Game Table», чтобы перенести участников в активную сцену.

Game Table хранит участников, инициативу, story points, публичные и GM-заметки. Ведущий управляет сценой, игроки видят доступную им часть.

## Режимы контента PublicSafe и PrivateFull

GenesysForge поддерживает два режима наполнения справочников.

- PublicSafe: безопасный публичный набор — названия, числа, структурные параметры, источники и краткие paraphrase-описания.
- PrivateFull: приватный режим для локального использования, где могут подключаться расширенные собственные материалы проекта.

При подготовке публичного деплоя используйте PublicSafe и не добавляйте оригинальные тексты из официальных книг в seed, справочники или пользовательскую документацию.

## Экспорт, импорт и печать

Лист персонажа можно экспортировать в JSON и импортировать обратно. Это удобно для резервной копии или переноса между инстансами.

Печатный лист открывается из карточки персонажа. В браузере можно распечатать страницу или сохранить её как PDF.
